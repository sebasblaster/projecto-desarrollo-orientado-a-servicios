Pasos para crear el proyecto desde cero:

1.) abrir la carpeta general del proyecto (capa de servicios y clientes) y abrir una nueva terminal para ejecutar:
npm create vite@latest app3_client_react -- --template react

- selecionar en las opciones:
✓ Select a framework:
  React

✓ Select a variant:
  JavaScript

✓ Use rolldown-vite (Experimental)?:
  No

✓ Install with npm and start now?
  Yes

2.) Entrar a la carpeta
cd app3_client_react

3.) Instalar dependencias
npm install

4.) Arrancar el cliente  (usar para arrancar el proyecto en la terminal de este cliente)
npm run dev

5.) Vite  mostrará una URL tipo:
http://localhost:5173/ → ahí vemos el template inicial de React.

6.) npm install bootstrap

7.) npm install react-google-recaptcha
